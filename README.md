# Install

## Windows
`xcopy Copilot.xml %APPDATA%\QtProject\qtcreator\styles`

## MacOS
`cp Copilot.xml ~/.config/QtProject/qtcreator/styles/`

## Linux
`cp Copilot.xml ~/.config/QtProject/qtcreator/styles/`
